3. **Activity Feed** : chips de filtre masquent les items dont `data-type` ne correspond pas au filtre actif. Bouton "Charger plus" revele les `.initially-hidden` et se cache ensuite.
4. **SPA navigation** : naviguer via la sidebar puis revenir sur les pages — verifier que les 3 inits sont rappeles via `reinitComponents()` (anti-double-bind `dataset.bound` protege contre les doubles inits).

## Points d'attention pour le tester

1. **Thumb position** : le deplacement du thumb est pilote par CSS via `[data-mode="light"]` sur `<html>`. Le tester doit verifier visuellement que setter `data-mode="light"` sur html deplace le thumb vers la droite.
2. **aria-checked semantique** : `aria-checked=true` signifie mode light actif. Counter-intuitif mais correct : le switch represente "est en mode clair ?".
3. **Disabled** : HTML natif gere via `disabled` sur `<button>` — pas de logique custom.
4. **Utilities typographiques** : .page-title et .section-title n'ont pas de responsive breakpoint — utilisent les tokens --type-* fixes. Si variation responsive souhaitee, issue de suivi.

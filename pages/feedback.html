Usage

Utiliser les alertes pour des messages persistants contextuels lies a l'etat de la page (confirmation d'action, avertissement systeme). alert-info pour les informations neutres, alert-success pour les confirmations, alert-warning pour les avertissements non-bloquants, alert-danger pour les erreurs critiques. Ne pas utiliser des alertes pour des notifications ephemeres — preferer les toasts. Ne pas afficher plus de 2 alertes simultanement sur une meme page.

HTML
<!-- Alert info --> <div class="alert alert-info"> <span class="alert-icon">&#8505;</span> <span>Nouvelle version disponible.</span> </div> <!-- Alert success --> <div class="alert alert-success"> <span class="alert-icon">&#10003;</span> <span>Deploiement reussi.</span> </div> <!-- Alert warning --> <div class="alert alert-warning"> <span class="alert-icon">&#9888;</span> <span>Utilisation elevee.</span> </div> <!-- Alert danger --> <div class="alert alert-danger"> <span class="alert-icon">&#10007;</span> <span>Echec de connexion.</span> </div>

Tokens status (fg / bg / border)

Triplets semantic pour Alert, Input (etat erreur), Button (variant danger), banners de lock. Contraste WCAG AA valide en mode light sur --surface-solid.

Apercu
warn — Avertissement non-bloquant
error — Erreur bloquante
info — Information neutre
success — Confirmation

Chaque status expose 3 variantes : -fg (texte/icone), -bg (fond teinte), -border (bordure). Tokens disponibles : --status-{warn,error,info,success}-{fg,bg,border}.

FAB — Floating Action Button

Bouton d'action flottant en position fixe bottom-right. 3 variantes : simple, mini, et menu radial expand.

FAB simple

Classe .fab — icone + hover scale + glow accent

Mini FAB

Classe .fab-mini — 40px vs 56px standard

FAB extended

Classe .fab-extended — pill avec label texte

FAB menu radial (expand)
Partager
Modifier
Supprimer

Clic sur le FAB pour ouvrir/fermer — icone tourne 45deg, actions en stagger

Usage

FAB fixe en production : .fab avec position: fixed; bottom: 1.5rem; right: 1.5rem;.

Menu expand : .fab-menu + .fab-trigger + .fab-actions. JS : initFAB() gere le toggle .open.

Variantes couleur : .fab-success, .fab-danger.

Fermeture automatique au clic exterieur.

Confirm Popover

Popover de confirmation inline avec 4 positions. Fermeture clic exterieur et Echap.

Position top (defaut)
Confirmer la suppression ?
Cette action est irreversible. L'element sera definitivement supprime.
Position bottom
Reinitialiser les parametres ?
Les preferences seront remises aux valeurs par defaut.
Position right
Archiver cet element ?
Il ne sera plus visible dans la liste active.
Usage

Structure : .popover-confirm-wrap > [data-confirm-trigger] + .popover-confirm.popover-confirm--top/bottom/left/right.

Interne : .popover-confirm-title + .popover-confirm-desc + .popover-confirm-actions ([data-confirm-cancel] / [data-confirm-ok]).

JS : initConfirmPopover() — toggle au clic trigger, fermeture Echap + clic exterieur.

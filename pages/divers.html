Accordion

Sections repliables pour FAQ ou documentation.

Quelle stack est utilisee ?▼
Next.js 14 (App Router) + TypeScript en frontend, FastAPI en backend, PostgreSQL + Prisma pour la base de donnees, Caddy en reverse proxy et Docker pour l'isolation.
Comment deployer un projet ?▼
Utilisez la commande /deploy dans Claude Code. Le skill deploy-caddy configure automatiquement le sous-domaine, le certificat HTTPS et l'auth gate.
Comment personnaliser la charte ?▼
Modifiez les variables CSS dans le bloc :root. Les couleurs, gradients, rayons et ombres sont tous parametrables via ces tokens.
L'auth gate est-elle obligatoire ?▼
Oui, par defaut tous les projets deployes sur *.msyx.fr sont proteges. Pour la desactiver, retirez le bloc basic_auth du Caddyfile et rechargez Caddy.

Bottom Navigation

Barre de navigation mobile fixee en bas, avec icones SVG et labels. Visible uniquement sur mobile (< 768px) en production — affichee dans un mockup telephone pour la demonstration.

5 items — defaut

Contenu de la page

5 items avec item actif en accent. Clic pour changer l'item actif.

3 items — minimal

Contenu de la page

Version minimale a 3 items.

Avec badge de notification

Contenu de la page

Badge numerique et badge point sur les items.

Usage en production (mobile only)

Integration dans un vrai projet :

<nav class="bottom-nav">...</nav>

La classe .bottom-nav est automatiquement cachee sur desktop via media query @media (min-width: 768px).

Utilisez .bottom-nav--demo pour forcer l'affichage en demonstration (dans un mockup).

JS : initBottomNav() gere le toggle actif automatiquement.

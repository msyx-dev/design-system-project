# os-livedemo

Design system vivant de msyx.fr — source de vérité pour tous les composants UI (cards, boutons, badges, formulaires, calendriers, login, etc.).

## Stack

- HTML/CSS/JS statique, composants partagés via `shared/`
- Thème dark (#0a0f1e), accent bleu #3b82f6, gradients bleu-violet
- Typo : Space Grotesk + Inter + Fira Code
- Servi par Caddy (pas de Docker)

## URL

https://os-livedemo.msyx.fr

## Déploiement

Fichiers servis directement par Caddy depuis ce répertoire. Aucun build nécessaire.
